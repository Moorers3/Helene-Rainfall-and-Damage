# Helene-Rainfall-and-Damage
## This is a map of rainfall totals and major road damage from Hurricane Helene in Counties of Western North Carolina
# Project Description
## This project contains multiple functions, libraries and data points. Including Font Awesome Icons on each data point of major road damage while infering home and building damage in surronding locations of the point. Using GeoJson I was able to pull data from sources and physically type in each data point for both polygons and points to create the county boundaries as well as markers for data. 
# Libraries and Functions
## Leaflet, JQuery, Font Awesome, NCOneMap, Carto DB, and GeoJson
# Sources/ Data
## The Charlottes Observer, WLOS, WRAL, USA Today, CBS 17, NCDOT, NOAA NHC, and North Carolina Emergency Management.
